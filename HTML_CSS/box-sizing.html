Khi chúng ta tạo phần tử có size được set và thêm cho nó các thuộc tính border, padding thì sẽ khiến size của phần tử thay đổi

box-sizing: content-box ; (mặc định)

Lúc này width = width + border (right-left) + padding (right-left)

height = height + border (top-bottom) + padding (top-bottom)

Để fix vấn để này ta sử dụng box-sizing: border-box ;

Nó sẽ tính làm sao để cho size phân tử giống như đã đc set ( thu nhỏ content lại chừa chỗ cho padding, border)
